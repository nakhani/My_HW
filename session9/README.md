# Session 9

In this session, I wrote a **Telegram Bot** which has below features:

 - *Play Game* 
 - *Create GIF* 
 - *Calculate your age* 
 - *Calculate MAX number* 
 - *Convert text to voice* 
 - *Calculate ARGMAX number*

