## Usage


**To run a GCM calculator  :**

```
python GCM.py
```

**To run a LCM calculator  :**

```
python LCM.py
```

**To run a Hangman Game  :**

```
python hangman.py
```

**To run a Random Number producer  :**

```
python randnum.py
```


**To run a Snake Game  :**

```
python snake.py
```

**To run a list sorter  :**

```
python sorted_list.py
```
