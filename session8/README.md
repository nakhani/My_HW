## Usage

**First install needed libraries :**
```
pip install -r requirements.txt
```

**To run Translator Program :**

```
python translate.py
```
**To run GIF creator :**

```
python gif.py
```
**To run Convert text to voice :**

```
python convert_text_to_voice.py
```
